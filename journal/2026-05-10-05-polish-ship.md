## LEARNINGS

- WSL can resolve Windows `npx` first, which breaks Newman path handling against a UNC current directory. The Makefile now uses native `newman` only if it is not under `/mnt/c`, otherwise it runs Newman inside `node:20-alpine`.
- CI needed `.env` mode `600` before `make up`; otherwise the preflight correctly refuses to boot.
